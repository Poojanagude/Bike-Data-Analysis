# 🚴 Bike Sales Analysis Dashboard

# Project Overview

This project analyzes global bike sales data using Tableau to uncover business insights related to revenue, profit, customer demographics, product performance, and regional sales trends.

The dashboard was designed to support data-driven decision-making through interactive visualizations and KPI tracking.

---

#Business Objectives

* Analyze revenue and profit performance
* Identify top-performing products and regions
* Understand customer purchasing behavior
* Monitor Year-over-Year sales growth
* Create an interactive executive dashboard

---

##  Key KPIs

* Total Revenue: $15.2M+
* Total Profit: $5.9M+
* YOY Growth: 101.87%


## Dashboard Insights

* Certain regions contributed significantly higher sales
* Product categories showed varying profitability
* Customer purchasing behavior differed by age group and gender
* Revenue trends indicated strong year-over-year growth
